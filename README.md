# Prime Kozmetik Mobil Katalog

Bu klasör GitHub Pages üzerinde yayınlanmaya hazırdır.

## Yayınlama
1. GitHub'da yeni bir repository oluşturun.
2. Repository adını `prime-kozmetik` yapın.
3. Bu klasördeki tüm dosyaları repository ana dizinine yükleyin.
4. Settings → Pages bölümüne gidin.
5. Source: Deploy from a branch
6. Branch: main / root
7. Save
